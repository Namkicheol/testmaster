**1. Krashen’s Input Hypothesis (Innatism)**

- 2언어 학습에서 input (=exposure) 가장 중요! > 노출을 받아들이는 것 (intake=internalize)이 중요함

- 영어를 잘하는 것은 output을 통해 확인할 수 있음

- 2언어 습득 과정: input > intake > output (이중 intake, output은 저절로 develop됨 > input이 가장 중요!)

- natural approach (저절로 익히게 된 모국어처럼 제2언어도 자연스레 익힐 수 있다): comprehension-based learning/approach (listening, reading 중심)

- 2개의 언어 학습 상황 (natural communication 상황, formal instruction 상황) 중 natural communication에서 input 제공하자고 주장함 > 2언어에 대한 지식을 저절로 얻게 됨 (acquired knowledge)

제2언어의 성공적 학습 조건:

1. natural communication 상황 속에서 **comprehensible input** (이해가능한 언어적 입력: (학습자 수준 i) + 1)을 주자

2. affective filter: 학습자가 comprehensible input을 받아들이기에 affective filter이 낮아야 한다

- affective filter: 새로운 언어 학습을 하다가 생기는 anxiety (불안함), stress, inconfidence, low motivation일수록 filter이 높아짐

> comprehensible input이 affective filter을 거쳐서 LAD의 intake에 하나씩 쌓임 (쌓이는 순서: 언어가 가지는 자연적 발달 순서(natural order; 학습자가 받아들일 준비가 되어있는 순서대로))

- 문법 수업 (formal instruction)은 언어 발달에 별로 큰 효과가 없다고 봄 - error correction이나 grammar instruction은 zero option(가르치지 말자!)

key terms: 1. **acquisition learning hypothesis** (학습 상황) 2. **input hypothesis** (학습 조건; comprehensible input 강조) 3. affective filter hypothesis (comprehensible input이 온전히 intake화 될 수 있도록 하는 조건 중 하나인 low affective filter 설명) 4. natural order hypothesis (intake에 해당 목표에 대한 지식이 차근차근 쌓이는 순서 설명) 5. monitor hypothesis (쌓인 언어를 가지고 저절로(emergent) 발화가 출현함; 완벽히 습득하지 않은 상태에서 언어 사용할 때 범하는 오류에 한해 바로잡아주는 역할은 모니터)

- **theory**: proved (증명됨) vs. **hypothesis**: 100% 증명되지 않아 아직도 연구 중

- 각 가설 꼼꼼히 정리 + classroom application 보고, 수업을 진행하기 위해서 어떤 가설을 기반으로 어떤 활동/교실 처치를 준비했는지 알 수 있어야 함 > 교실 수업 시사점 정리

\*영어 수업 중 comprehensible input hypothesis를 토대로 structural syllabus 선택하여 수업 진행함. 그런데 수업이 생각보다 잘 진행되지 않음. 문제점은? > comprehensible input hypothesis를 토대로 수업하려고 한다면 정해진 형태의 문법 수업은 성공적 언어 학습을 이끌어내지 못한다는 natural order hypothesis를 위배함.

\*문법 수업과 관련해서 natural order hypothesis 몇 번 기출됨

- **non-interface position**: learnt knowledge가 절대 acquired knowledge가 될 수 없음 > 교실 상황에서는 acquired knowledge만 키워줘야 함 by natural communication, meaning-focused

**2. McLaughlin’s Information processing model (Cognitivism)**

- 2언어 학습 과정 중에 얻게 되는 지식이 인지구조 안에서 어떻게 발전되어서 실제 언어사용까지 연결되는가? 인지구조 안에서의 언어 습득 설명

- 2언어 학습 과정 중 얻게되는 2언어 지식을 ‘information’이라고 칭함

- ‘intake’, 즉 2언어 습득 과정에 초점을 둠

- controlled / automatic processing

- controlled processing: 처음 target form/rule을 배웠을 때의 일시적 능력

- automatic processing: 절대 잊어버리지 않는 long-term memory 안에 지식이 들어감

- 교실 수업(language classroom)은 short-term memory에서 long-term memory로 갈 수 있도록 도와야 함 > target form에 대한 meaningful practice 해야 함 (=form-focused instruction)

> 교실의 meaningful practice를 거쳐 controlled processing이 automatic processing화 될 수 있다

- interface position (Krashen과 반대): 교실에서 meaningful practice를 통해서 2언어 학습에서 갖게 된 지식은 영구적 지식으로 바뀔 수 있다

**Bialystok’s Implicit and Explicit Knowledge Model (Cognitivism)**

- 2언어 학습 과정 중 얻게되는 2언어 지식을 ‘knowledge’라고 칭함

- implicit / explicit linguistic knowledge

> 교실의 meaningful practice를 거쳐 explicit linguistic knowledge가 implicit linguistic knowledge로 발전할 수 있다

\*language use 측면에서 2언어 학습에 대한 지식을 분류할 때, explicit/implicit linguistic knowledge, controlled/automatic processing은 인지구조 안에 있는 지식 상태를 설명하는 용어, 실제적으로 언어를 사용하도록 하는 language use 측면에서 목표 언어에 대한 지식을 설명할 때 사용하는 언어는 **declarative/procedural knowledge**

- (Krashen) learnt knowledge & (Bialystok) explicit linguistic knowledge & controlled processing & declarative knowledge > 전부 같은 ‘일시적인 능력의 형태로 실제 target form에 대해 설명할 수 있는 지식’을 의미함. formal instruction을 갖게 되는 지식. 맨 처음 갖게 되면 controlled processing (McMcLaughlin)에 머물게 됨.

- acquired knowledge & implicit linguistic knowledge & automatic processing & procedural knowledge > natural communication에 의해 저절로 얻게된 지식. 머리 속에 들어있음.

> Krashen: 이 둘이 하나로 합쳐지지 않기 때문에 (non-interface position) formal instruction에서 얻게된 learnt knowledge에 초점을 맞추는 수업을 할 필요가 없다 > meaning-focused natural communication 강조

> 인지주의자: learnt knowledge/explicit knowledge도 acquired knowledge/implicit knowledge가 될 수 있다 (interface position). 교실 안에서의 form-focused instruction의 meaningful practice 통해서 가능.

- 2언어 학습 과정 중에 갖게 되는 지식의 현 상태를 일컫는 용어: learnt/acquired knowledge, explicit/implicit knowledge

- 실제 교실 수업 안에서 언어를 사용하느냐 사용하지 않느냐에 대해, 실제로 그 지식이 practical하게 사용되는 측면의 언어: declarative/procedural knowledge > 직접적인 classroom teaching과 더 관련 있음 > 중요!

- language use 측면에서 해당 목표에 대한 지식을 가지고 있는 상태, 이를 설명할 수 있는 지식이 declarative knowledge / 이를 직접 사용할 수 있도록 하는 것은 procedural knowledge

> 시사점: “모든 수업의 공통적 목표는 declarative knowledge를 procedural knowledge화 시키는 것이다”

**[Error Analysis (오류 분석과 학습자 언어)]**

- **interlanguage** (=learner language): an intermediate system located somewhere between the learner’s native language and the target language; 모국어와 목표어의 중간에 있는 불완전한 언어 체계

- interlanguage 특징: 1. systematic 2. variable 3. dynamic

- interlanguage 발달 단계: 1. **presystematic stage** 2. **emergent stage (**\***backsliding**) 3. **systematic stage** 4. **stabilization stage** (\***fossilization**)

- **U-shaped Learning**: 1. correct form (controlled processing) > 2. incorrect form (backsliding /overgeneralization /backsliding) > 3. correct form (automatic processing)

- **Error Analysis**: 학습자의 오류를 목표어의 체계에 도달하는 하나의 과정(interlanguage)으로 인식하고 학습자의 오류를 관찰, 분석, 시정함

- Error 원인: 1. **interlingual transfer(=interference**) 2. **intralingual transfer** (\***overgeneralization**)

- Error 종류: 표면적 특성의 범주 – 1. **addition** 2. **omission** 3. **substitution** 4. **misordering** - 의사소통의 흐름에 따른 범주 – 1. **global error** 2. **local error**

- 오류의 역할, 수정 범위 (수정 기준)

- **Feedback**: negative feedback은 학습자가 자신의 발화를 돌아보고 정확한 발화를 사용하도록 유도함으로써 2언어 습득 가속화, fossilization 현상 방지함

- Feedback 종류: 1. **Explicit correction** 2. **Recasts** (**negative evidence** vs. **positive evidence**) 3. **Clarification requests** 4. **Metalinguistic feedback** 4. **Elicitation** 5. **Repetition**

**[Discourse Analysis (담화 분석) – Constructivism]**

- **Halliday’s Seven Language Functions**: 언어 기능 중심의 notional functional syllabus의 토대, 언어 교수에서 의사소통적 기능 강조 > 1. instrumental 2. regulatory 3. representational 4. interactional 5. personal 6. heuristic 7. imaginative functions

- 담화의 구성 요건:
1. **cohesion**: connectivity in a text; grammatical – conjunction, reference, substitution, ellipsis; lexical – repetition, synonyms, superordinates, general words
2. **coherence**: how easy it is to understand the writing; 논리적 일관성; cohesive device 없어도 한 주제에 대해 문장들이 논리적 관계를 이루고 있으면 coherent text 될 수 있음
\*cohesion은 coherence를 보완하는 요소; cohesion 있다고 해서 반드시 coherence 있는 텍스트인 것은 아님

- 대화 분석 (**Conversation Analysis**):
1. **Attention getting** 2. **Topic nomination** 3. **Topic development** (**turn taking**, **topic clarification**, **topic shifting** (**gambit strategy**), **topic avoidance**) 4. **Topic termination**

- Question Types: 1. **Display questions** (closed) 2. **Referential questions** (open)
 - Effective questioning (what are effective questions? question modification devices – **syntactic modifications**, **semantic modifications**, **lexical modification**)
 - classroom discourse: **IRE** (initiation (assessing questions), response, evaluation), **IRF** (initiation, response, feedback (assisting questions))

**Chap 5. Styles and Strategies-Based Instruction (SSBI)**

- The Procedure of SSBI:
 1. **Strategy preparation** 2. **Strategy awareness-raising** 3. **Strategy training** 4. **Strategy practice** 5. **Personalization of strategies**

- Characteristics of Learner Strategies:
 1. Strategies are conscious behaviors involving cognitive, social, and affective processes
 2. Cognitive, social, and affective strategies are controlled by a higher level of cognitive processes known as metacognitive processes.
 3. Although the use of strategies requires attention on the part of the learners, not all the tasks require the same amount of attention.
 4. 전략은 개별 학습 시 사용되기도 하나, 의사소통 및 더 나은 학습 결과를 성취하기 위해 복합적으로 사용되기도 한다.
 5. 언어 학습에 도움이 되는 전략이 있고 도움이 되지 않는 전략이 있다.
 6. 전략 사용은 내적 및 외적 요인(학습자 자신, 과업, 환경)에 의해 영향을 받는다.
 7. 하나의 종합적인 전략(general strategy)에는 관련된 여러 개의 세부적인 전략(tactics)이 포함된다.
 8. Learners’ knowledge about and use of strategies must be understood in its interaction and sociocultural contexts.

- Techniques for Teacher Strategies:
 1. **Teacher modeling** (=**think aloud technique**)
 2. **Awareness raising**
 3. **Guided practice**
 4. **Review and reuse**
 5. **Reflect and refocus**

- Use of Compensatory Techniques:
 1. 과도한 field-independent/left-brain dominant learners: integrative language techniques, retelling stories, skimming task
 2. 과도한 field-dependent/right-brain dominant learners: syntactic or semantic clues searches, scanning for specific information, information-gap techniques, proofreading
 3. 과도한 low ambiguity tolerance: brainstorming, retelling stories, role-play, skimming task
 4. 과도한 impulsive learners: making inference based on evidence, syntactic or semantic clue searches, inductive rule generalization based on evidence
 5. 과도한 reflective learners: small group techniques, role-play, brainstorming, fluency techniques

**[Syllabus]**

- **structural syllabus** – 차시를 나누는 기준: forms and structures, usually grammatical
 - e.g.) nouns, verbs, adjectives, statements, questions, complex sentences, subordinate clauses, past tense, pronunciation, morphology
 - **audiolingualism** (specific structured dialogues are drilled and memorized to conceptualize key structures), **silent way**

- **notional-functional syllabus** – 차시를 나누는 기준: functions performed when language is used
 - e.g.) informing, agreeing, apologizing, requesting, promising
 - **notions**: e.g.) size, age, color, comparison, time

- **situational syllabus** – 차시를 나누는 기준: real or imaginary situations
 - e.g.) seeing the dentist, buying a book at the book store, meeting a new student, asking directions in a new town > specific situations
 - c.f.) **topical/thematic syllabus**: grouping lessons around a topic e.g.) topic: community – situation: at the post office, 동사무소, etc.
 - **situational dialogue approach** (meaningful conversational interchange in specific contexts; the main focus is communicative effectiveness within a given situation instead of language structures)

- **task-based syllabus** – complex and purposeful tasks to be performed with the target language
 - c.f.) notional-functional syllabus (차시 나누는 기준 function) 와 차이점: 차시 나누는 기준이 task (한 task 내에서 여러가지 function 활용 가능)
 - strong form vs. weak form

- **product-oriented syllabus**: focus on things learned at the end of the learning process (outcomes)
 - e.g.) structural syllabus (목표 문법), situational syllabus (목표 상황), notional-functional syllabus (목표 기능)

- **process-oriented syllabus**: focus on skills and processes involved in learning language
 - procedural/task-based syllabus (task는 정해져 있지만 그 안에서 어떤 표현 쓸지 모름), learner-centered syllabus (학생이 정함), content syllabus (content는 정해져 있지만 학생에 의해 사용되는 전략, 스킬 달라짐)

- **adding**: 양적, 질적으로 자료 추가
 - **extending**: 양적 추가 (같은 내용, 같은 수준)
 - **expanding**: 질적 추가 (다른 기능, 다른 차원의 언어 접근법)

- **deleting**: 양적, 질적으로 자료 생략/삭제
 - **subtracting**: 양적 생략
 - **abridging**: 질적으로 다른, 불필요한 활동 생략

- **modifying**: 학습자의 요구 / 언어 접근법 / 특정 교수법 (특히 CLT)에 적절하도록 형식 수정
 - **rewriting**: 텍스트 내용 수정
 - **restructuring**: 교실 / 그룹 배치 수정

- **simplifying**: 언어 수준 조절 to make them more accessible to learners
 - **sentence structure**: sentence length is reduced / complex sentences rewritten into simpler ones
 - **lexical content**: the number of new vocabulary
 - **grammatical structure**: passives converted to actives, simple past tense to simple present

- **reordering**: 활동 / 자료 제시 순서를 더 적합하게 변경

**[Material evaluation]**

- **external evaluation**: 객관적 사실 바탕으로 분석 – impressionistic method
 - general impression; wide-ranging but relatively superficial
 - e.g.) glancing, skimming on the cover, the introduction, and table of contents
 - analysis level 1 (what is there)

- **internal evaluation**: 의미 분석 – in-depth method
 - analysis level 2 (what is required of users), 3 (what is implied)

**[Computer-Assisted Language Learning]**

- Corpus linguistics in the classroom:
 - **data driven learning** - **concordancing program** (=**concordancer**): e.g.) to analyze examples of learner’s typical error patterns
 - **corpora** (sing. **corpus**) of texts by native speakers: e.g.) examples for patterns of the target language
 - e.g.) learners can use a **concordancer** and **corpora** to find examples and develop their own rules for grammatical structures, idioms, and general usage, e.g.) investigating the difference between “look” and “see”; to look up authentic examples of language patterns

- **Computer-Mediated Communication (CMC)**:
 - **asynchronous mode**: time gap between communicators e.g.) e-mail, bulletin board, keypal project
 - **synchronous mode**: e.g.) chat programs, video conferencing
 - **MOO** (Multi-user Object-Oriented dimension): an online community (asynchronous + synchronous)
 - **e-learning**: learning that takes place using technology such as the Internet and portable devices like mobile phones or MP3 players.
 - **distance learning**: like the term ‘e-learning’, an umbrella name for the terms below
 - **open learning**: one aspect of distance learning; the more open a distance course, the more autonomy the learner has in deciding what course content to cover, and how and when to do so
 - **online learning**: a facet of e-learning; takes place via the Internet
 - **blended learning**: mixture of online and face-to-face course delivery
 - **Computer-Adaptive Testing (CAT)**: match the difficulty of each test item (=item facility) to the abilities of the learners being tested; the following question can be tailored or adapted

- **digital literacy learning**: consists of computer literacy, information literacy, knowledge literacy
 - self-directed learning
 - **digital literacy**: the ability to understand and use information in multiple formats from a wide variety of sources when it is presented via computers, particularly through the medium of the Internet
