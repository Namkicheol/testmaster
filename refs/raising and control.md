Believe 동사 ( consider, know, prove ,report)
1) Believe ( ECM ) / S-O raising  / 1차 분석, 2차 분석 / + TP

- Believe: two place predicate   Ex) Ed believes the story. ( One external argument and One internal
argument )
>> Both 'Ed' and 'the story' has thematic relation to the predicate 'believe'.

* Ed believes the story to be false : 다음 문장에서 the story는 어디에 위치하는가?
답 : Direct object of the verb believe ( Believe + NP + to infinitive ) : 1차 분석
1) The story is believed to be false by Ed.  ( main passivize가 되는 걸 보면 okay )
2) Ed believes him to be a traitor. ( the story가 있는 자리에 대명사가 들어갈 수 있는 걸 보면 the story가 목적
어 자리라는 것을 알 수 있다 )

그러나 이러면 문장의 의미가 ' Ed believes the story '가 되어 버린다. 그러나 이 문장은 그런 뜻이 아니다.

- Ed believes the story to be false : 다음 문장에서 the story는 어디에 위치하는가?
답 : subject of the to-infinitive clause : believe + ( NP to VP ) : 2차 분석

1) idiomatic meaning과 expletive and pleonastic it은 주어자리에 있을 때에만 의미를 갖는다.
Ex) Ed believes it always to be raining in London.
Ed believes there to be a traitor in the company
Ed believes the coast to be clear.
Ed believes the fat to be in the fire.

모두 말이 되는 것으로 보아 NP인 it과 there은
지금 목적어 자리가 아니라 하위절의 주어

2) Post passivize가 된다는 점 ( subordinate clause passivize )
Ed believes ( the jury to have given the wrong verdict )
Ed believes ( the wrong verdict to have been given to the jury )
: believe는 지금 바로 뒤에 있는 NP에 theta-role을 주고 있지 않고 뒤에 있는 전체에 theta-role을 주고 있다. 따라서
하위절의 수동태 두 문장은 서로 같은 의미이다.

- what cleft 가능 ( ONLY TP can be in the focus position of pseudo cleft sentence )
What Ed believed was the story to be false.
- for 쓰면 불가능
I have never known for the prime minister to lie.

* ECM 동사들의 특징 : Allow only single clausal complement.
Ex) John believes[ Bill ] [that he has slept]. >> Two complement >> 비문
      >> John believes Bill to have slept. >> One complement >> 정문

Expect 동사

2) Expect ( ECM / Control ) / S-O raising , subject control / + CP and TP

- Expect : Two place predicate
SC: George expects to win the game. ( CP )
SOR : George expects Sue to win the game ( TP ) : believe와 유사
            We expected Brian to leave. ( TP )

- SOR ( Two place predicate ) / ECM
We expected Brian to leave : Two arguments : We, Brian to leave
>> There is no thematic relation between expected and Brian. Brian is not the object of expected but a
subject of to leave. : Brain을 하위절의 주어로 보는 경우

1) Expletive :
We expected it to rain there.

3) post passivization
Barbara expected Megan to feed the dog.
Barbara expected the dog to be fed by Megan.

2) 뒷문장을 전체로 주어로 만들기 가능
Barbara expected Megan to feed the dog.
For Megan to feed the dog was expected by Barbara.

Brian을 상위절의 목적어로 보는 경우

1) Main passivization 가능
Barbara expected Megan to feed the dog.
>> Megan was expected by Barbara to feed the dog.

* ECM 동사의 경우 only single clausal complement를 취할 수 있다. It does not allow two internal arguments.
He expected that Sue went to the audition.
* He expected Sue that she went to the audition.
He expected it to be sunny.
He expected there to be a lot of people at the party.

- Pseduo cleft focus position ECM인 경우에만 가능
What George expected was Sue to win the game.
* What George expected was to win the game.

Want 동사 ( like, intend )

3) Want ( ECM / SOR ), SC / 근데 believe랑 조금 다름 / + CP and TP
: ' want ' assigns two theta roles ( Agent or experiencer to external argument and a proposition )
: When the NP raises to the object position of the main clause, 'want' assigns accusative case to it.

- Want 가 believe랑 비슷하다는 경우
1) idiomatic meaning의 가능성
: Jean wants there to be a ceasefire between Lily and Tom.

2) Post passivization 가능성
: Jean wants Robert to be examined by the doctor.
= Jean wants the doctor to examine Robert.

- Want가 believe랑 다르다는 경우 ( Jean wants ( for ) Robert to go over there ) 눈에 안보이는 for가 case 줌 )
1) main clause passivization이 불가능
: Jean wants Robert to go over there.
>> * Robert was wanted by Jean to go over there.

( like가 want와 같은 유형 )
: The head minister liked all parenst to visit the school.
The head minister liked the school to be visited by all parents.
* All parents were liked to visit the school.

SC: Jim wanted to leave London.

I never intended that to happen.
I never intended for that to happen.

persuade 동사 ( convince , order )

1) Persuade ( OC ) + CP
Persuade NP to VP : NP는 항상 상위절의 목적어

1) expletive 허사, idiomatic meaning X
* Ed persuaded it to be hot in the room.
* Ed persuaded there to be a party.
* Ed persuaded the coast to be clear.
* Ed persuaded the fat to be in the fire.

2) post passivization불가능
Ed persuaded Brain to interview Melanie .
=/= Ed persuaded Melanie to be interviewed by Brian.
